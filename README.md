# ESP32 Smart Toll Gate System
##Description
An ESP32-based IoT system that automates toll gate operations using ultrasonic sensors for vehicle detection, servo motors for gate control, and LEDs for traffic and streetlight management. Controllable remotely through the Blynk mobile app for real-time monitoring.

## Features 
- Automated vehicle detection
- Smart gate control
- Traffic light system
- Streetlight automation
- Remote monitoring via Blynk app

##Technologies Used
- ESP32 microcontroller
- Arduino IDE
- Blynk IoT Platform
- Ultrasonic sensor
- Servo motor
- LEDs

##How to Run
1. Connect hardware components to ESP32
2. Install Arduino IDE and Blynk library
3. Update WiFi credentials and Blynk token
4. Upload code to ESP32
5. Configure Blynk app
